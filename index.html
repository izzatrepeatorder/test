How to read this dashboard
Target (10.0%) — the project goal from the repeat order audit. Every cohort should move toward this.
Baseline (Nov 2025: 7.0%) — permanent anchor. First full cohort before any campaigns. Never changes.
Latest (Jan 2026: 7.7%) — most recent month where full 90 days have passed. When positive, campaigns are working. When negative, we're below where we started.
